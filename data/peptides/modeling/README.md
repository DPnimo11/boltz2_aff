# Legacy BH3/p53 modeling features

This is the modeling-ready feature area for the earlier BH3/p53 Part 2 arm,
which is currently on hold. `features/boltz_embeddings/` contains the 2,139
previously extracted per-complex affinity embedding files. Experimental source
tables live under `../source/`, and generated Boltz inputs live under
`../boltz/inputs/`.

The active mutation study is `data/peptide_systems/`; these files are retained
because the BH3/p53 analysis may return.
