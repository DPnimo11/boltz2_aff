# Transferred Boltz reference run

These are inputs and execution metadata from the Boltz run transferred with
the Rognan/ULVSH paper data. They are reference provenance, not outputs from
this repository's modified affinity-embedding exporter.

- `inputs/<target>/<variant>/*.yaml` retains the transferred WT, mutant, and
  shuffled run inputs locally. The input tree is intentionally gitignored.
- `Boltz2.job` is the transferred job description.
- The six scalar affinity values from every result JSON are preserved in
  `../modeling/features/boltz_scalars.tsv`.

The original 5.7 GB output tree was removed after the scalar table was checked
record-for-record against all 2,830 JSONs. Its PAE/PDE arrays, structures,
confidence products, and `pre_affinity` intermediates were not generated by
the modified exporter and are not consumed by the current analysis.
